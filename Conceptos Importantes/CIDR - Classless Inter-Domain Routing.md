CIDR se refiere al **Enrutamiento entre Dominios sin Clases**, el cual es un método eficiente para asignar direcciones IP y enrutar paquetes IP, deseñado para reemplazar al antiguo sistema de clases y optimizar el espacio de direcciones IPv4.
