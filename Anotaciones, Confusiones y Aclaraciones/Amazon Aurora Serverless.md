
Cuando se trabaja con solo [[Amazon Aurora]], el servicio requiere que una instancia esté en ejecución todo el tiempo, lo cual es más costoso.

**Amazon Aurora Serverless** es una configuración bajo demanda y de escalado automático para Amazon Aurora. La base de datos se inicia, se detiene y escala la capacidad hacia arriba y hacia abajo automáticamente según las necesidades de la aplicación. Esta es una solución de base de datos ideal para aplicaciones usadas con poca frecuencia.

Originalmente, Amazon Aurora se lanzó como una base de datos donde tenías que elegir el tamaño de la instancia (CPU y RAM). Sin embargo, para responder a la necesidad de no gestionar infraestructura, AWS lanzó **Amazon Aurora Serverless**.
