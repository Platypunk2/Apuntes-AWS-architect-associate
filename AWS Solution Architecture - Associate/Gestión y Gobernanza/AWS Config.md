**Permite evaluar, auditar y evaluar las configuraciones de sus recursos de AWS. Supervisa y registra continuamente las configuraciones de recursos de AWS y permite automatizar la evaluación de configuraciones grabadas con respecto a las configuraciones deseadas.**

# Descubrimiento
AWS Config descubrirá los recursos que existen en la cuenta, registrará la configuración actual y capturará cualquier cambio en estas configuraciones.

# Gestión del Cambio
Cuando se crean, actualizan o eliminan recursos, AWS Config transmite estos cambios de configuración a Amazon Simple Notificatioin Service (SNS), para que se notifique de todos los cambios de configuración.
