**Un servicio de almacenamiento de caché en memoria completamente administrado**

Un servicio de almacenamiento en caché en memoria (base de datos) totalmente administrado que puede admitir casos de uso flexibles en tiempo real.

Proporciona Redes administrado o Memcached como servicio y se utiliza mejor para <u>leer cargas de trabajo pesadas</u>  con requisitos de baja latencia.

El uso de **ElastiCache** puede ayudar a reducir las cargas de trabajo de las bases de datos (costosas) y utilizarse para almacenar datos de sesión (servidores sin estado).

### Redis vs Memcached
**Un servicio de almacenamiento en caché en memoria completamente administrado**
 - **Memcached** es un sistema de almacenamiento en caché de memoria distribuida diseñado para facilitar su uso y simplicidad. Es muy adecuado como caché o almacén de sesiones.
 - **Redis** es un almacén de estructura de datos en memoria que ofrece un amplio conjunto de características. Es útil como caché, base de datos, agente de mensajes y encolamiento.
