**Servicio de integración de datos serverless (sin servidor) y totalmente administrado que facilita la extracción, transformación y carga (ETL) de datos para análisis, machine learning y desarrollo de aplicaciones.**

AWS Glue es un servicio administrado de **ETL (Extract, Transform, Load)** de AWS, diseñado para preparar y transformar datos para análisis, machine learning y desarrollo de aplicaciones.

Permite **descubrir, catalogar, limpiar, transformar y mover datos** entre distintos almacenes de datos, de forma serverless.
