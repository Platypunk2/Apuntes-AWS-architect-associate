**AWS Firewall Manager es un servicio de administración de seguridad que le permite configurar y administrar de forma centralizada reglas de firewall en sus cuentas y aplicaciones en AWS Organizations.**

# Caso de uso
- Proteger aplicaciones alojadas en instancias EC2.
- Implementar herramientas a escala para proteger los datos.
- Auditar continuamente los recursos.

# Conceptos clave

Amazon Firewall Manager permite implementar AWS Network Firewall de forma centralizada en todas las VPC ([[Amazon VPC]]). Los cambios se propagan automáticamente a través de sus cuentas y VPC.

Implementar automáticamente grupos de seguridad de VPC ([[Amazon VPC]]), reglas de AWS WAF ([[AWS WAF - Web Application Firewall]]), protecciones AWS Shield Advanced ([[Amazon Shield]]), reglas de AWS Network Firewall y reglas de DNS Firewall de resolución de Amazon Route 53 ([[Amazon Route 53]]).
