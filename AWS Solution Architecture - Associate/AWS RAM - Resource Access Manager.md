**Ayuda a compartir seguramente tus recursos de AWS con otra cuentas de AWS**

AWS Resource Access Manager (RAM) es un servicio que te permite compartir recursos de AWS de forma fácil y segura con cualquier cuenta de AWS o dentro de tu Organización de AWS. Puedes compartir recursos de AWS transit Gateways, subredes, configuraciones de AWS License Manager y reglas de Amazon Route 53 Resolver con RAM. RAM elimina la necesidad de crear recursos duplicados en varias cuentas, reduciendo la sobrecarga operativa de gestionar esos recursos en cada una de las cuentas que poseas. puedes crear recursos de forma centralizada en un entorno multicuenta, y utilizar la RAM para compartir esos recursos entre cuentas en tres sencillos pasos: crear un recurso compartido, especificar los recursos y especificar las cuentas. La RAM está a tu disposición sin coste adicional.
