**Amazon SageMaker es un servicio completamente administrado que proporciona a cada desarrollador y científico de datos la capacidad de construir, entrenar y desplegar modelos de aprendizaje automático rápidamente. Puede conectarse directamente con fuentes de datos y tiene algoritmos incorporados para facilitar el proceso de ML.**

Es la plataforma de **Machine Learning (ML) end-to-end** de AWS, diseñada para que científicos de datos e ingenieros puedan **construir, entrenar, optimizar y desplegar modelos de ML** a escala, sin gestionar infraestructura.

El ciclo de vida de ML es complejo: preparar datos, experimentar, entrenar, optimizar hiperparámetros, evaluar, desplegar y monitorear. SageMaker **unifica todo ese flujo** en un solo servicio.
