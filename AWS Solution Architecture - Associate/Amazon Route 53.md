**Una forma confiable y rentable de enrutar a los usuarios finales a las aplicaciones en Internet**

Conecta las solicitudes de los usuarios a la infraestructura que se ejecuta en AWS. Servicio web para el sistema de nombres de dominio (DNS) en la nube, altamente disponible y escalable.

**Simplicidad**
Amazon Route 53 Traffic Flow facilita la configuración de lógica de enrutamiento sofisticada para sus aplicaciones.

**Velocidad**
Mediante una red anycast global de servidores DNS en todo el mundo, Amazon Route 53 está diseñado para enrutar automáticamente a sus usuarios a la ubicación óptima dependiendo de las condiciones de la red.
