# Resumen
En pocas palabras, EC2 te permite alquilar servidores virtuales en la nube para ejecutar tus aplicaciones, dándote control total sobre el sistema operativo y el hardware virtual.

Lo beneficios son:
- **Control Total:** Tienes acceso de administrador (root) al sistema operativo.
- **Seguridad:** Integrado con VPC (Redes virtuales privadas) y Security Groups (firewalls virtuales).
- **Elasticidad:** Con **Elastic Load Balancing (ELB)**, puedes distribuir el tráfico entre múltiples instancias para que tu app nunca se caiga.
