**Como conectarse directamente a otras VPC**

- Conecta miles de VPC a través de cuentas.
- Conecta sus VPC y centros de datos a través de una única puerta de enlace.
- Centraliza las conexiones VPN y AWS Direct Connect.
- Controla las segmentaciones y el flujo de datos con tablas de enrutamiento
- Diseño de Hub and Spoke.
- Hasta 50 Gbps por conexión VPC (incrementable).
