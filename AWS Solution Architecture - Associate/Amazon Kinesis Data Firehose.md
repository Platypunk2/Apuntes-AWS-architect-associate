**Servicio administrado de AWS para cargar datos de streaming en tiempo real hacia destinos de almacenamiento y análisis, sin necesidad de gestionar infraestructura.**
### ¿Qué hace?

Captura, transforma y entrega datos en streaming de forma continua hacia destinos como:

- **Amazon S3**
- **Amazon Redshift**
- **Amazon OpenSearch**
- **Splunk**
- **Datadog**, New Relic, y otros servicios de terceros via HTTP
