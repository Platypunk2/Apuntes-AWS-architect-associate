**AWS Shield es un servicio de protección de denegación de servicio distribuido (DDoS) administrado que protege las aplicaciones que se ejecutan en AWS.**

## Amazon Shield (Standard)

AWS Shield Standard defiende contra la mayoría de los ataques DDoS ([[Ataques DDoS]]) de red y capa de transporte que ocurren con frecuencia y que se dirigen a tu sitio web o aplicaciones. AWS Shield Standard, utilizado con Amazon CloudFront ([[Amazon CloudFront]]) y Amazon Route 53 ([[Amazon Route 53]]), proporciona una protección integral de disponibilidad contra todos los ataques de infraestructura conocidos (capas 3 y 4).

Servicio administrado, proporcionado sin cargo adicional.

## AWS Shield Advanced

AWS Shield Advanced proporciona mayores niveles de protección contra ataques dirigidos a tus aplicaciones que se ejecutan en recursos de Amazon Elastic Compute Cloud (EC2) ([[Amazon EC2]]), Elastic Load Balanced (ELB) ([[Elastic Load Balancing]]), Amazon CloudFront ([[Amazon CloudFront]]), AWS Global Accelerator ([[AWS Global Accelerator]]) y Amazon Route 53 ([[Amazon Route 53]]).

El costo incluye algunas características de WS WAD y AWS Firewall Manager.
